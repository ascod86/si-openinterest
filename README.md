# Si Open Interest — Суммарный открытый интерес

График суммарного открытого интереса по фьючерсу Si (доллар/рубль) на Московской бирже.

## Установка

**1. Установить Python** — python.org/downloads (при установке поставить галку "Add Python to PATH")

**2. Открыть папку в командной строке**
- Зайти в папку `si-openinterest`
- Shift + правая кнопка мыши → "Открыть окно PowerShell здесь"

**3. Установить зависимости** (один раз)
```
pip install requests pandas plotly
```

**4. Добавить API ключ**

Создать файл `token.txt` в папке со скриптом и вставить туда свой API ключ от [data.moex.com](https://data.moex.com/personal-account).

**5. Запустить**
```
python fetch_oi.py
```

Скрипт скачает данные (~30 сек при первом запуске) и откроет график в браузере.

## Повторный запуск

При следующих запусках докачиваются только новые дни — работает за секунды.

## Данные

- Источник: MOEX Algopack API (подписка на [data.moex.com](https://data.moex.com))
- Обновление: каждые 5 минут
- История: с 2024 года
- Таймфреймы: 1D и 4H (кнопки на графике)
