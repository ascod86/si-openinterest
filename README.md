# Открытый интерес USD/RUB · CNY/RUB — карточка

Аккуратная карточка с открытым интересом на MOEX и переключателем пар **₽/$ ↔ ₽/¥**.
Для каждой пары — **календарный** и **вечный** фьючерс **раздельно**:
- USD/RUB: `Si` (все календарные) и `USDRUBF` (вечный);
- CNY/RUB: `CR` (все календарные, в FORTS assetcode `CNY`) и `CNYRUBF` (вечный).

ОИ считается в контрактах, а у календарного и вечного разный размер контракта и ГО,
поэтому в один итог они не суммируются. Карточка показывает 2 последних торговых дня
и изменение ОИ от конца предыдущего дня к концу следующего. Обе пары строятся сразу,
переключатель на странице показывает нужную (JS, без сервера). Реалтайм через Algopack.

## Установка

**1. Установить Python** — python.org/downloads (галка "Add Python to PATH")

**2. Установить зависимости** (один раз)
```
pip install requests
```

**3. Добавить API ключ**

В файл `token.txt` рядом со скриптом вставить токен Algopack от [data.moex.com](https://data.moex.com/personal-account).

**4. Запустить**
```
python fetch_oi.py
```
Скрипт построит `si_openinterest.html` и откроет карточку в браузере.

## Автообновление

Карточка обновляется автоматически по cron каждые 30 минут (см. `crontab -l`):
```
*/30 * * * * python3 .../fetch_oi.py --no-open
```
- `--no-open` — не открывать браузер (для фонового запуска).
- Открытая вкладка сама перечитывает файл раз в 10 минут (`<meta refresh>`).
- Логи фоновых запусков: `data/cron.log`.

Сменить частоту или убрать автообновление: `crontab -e` / `crontab -r`.

## Данные

- Источник: MOEX Algopack API (подписка на [data.moex.com](https://data.moex.com)), реалтайм.
- Активы: `Si` (агрегат всех календарных контрактов) и `USDRUBF` (вечный фьючерс) — раздельно.
- ОИ за день берётся по последнему срезу (концу торговой сессии), в контрактах.
- Изменение в деньгах (в скобках) = Δконтрактов × ГО на контракт. ГО берётся из FORTS
  (`/iss/engines/futures/markets/forts/securities`): для Si — фронтовый контракт с макс. OI.
